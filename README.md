# Screen Time Tracker 📊

Pet-проект для автоматического сбора, хранения и анализа статистики использования приложений на ОС Windows. Проект реализует полноценный локальный процесс работы с данными (Data Pipeline): от сбора сырых логов операционной системы до агрегации и визуализации бизнес-метрик.

## 🚀 Текущий функционал

* **Фоновый сбор данных (Data Collection):** Скрипт `main.py` отслеживает активное окно пользователя через Windows API (`win32gui`, `psutil`), работая в фоновом режиме.
* **Событийное логирование (Time Series):** Данные записываются в реляционную БД (`SQLite`) только в момент смены активного окна. Фиксируется процесс, заголовок окна, время начала, конца и длительность сессии в секундах.
* **Умная очистка данных (ETL & Mapping):** Системные `.exe` файлы переводятся в читаемые названия через `config/process_names.json`. Если запущен неизвестный процесс, скрипт автоматически очищает его название от расширений и цифр, добавляет маркер `*` и сохраняет в JSON для дальнейшего ручного ревью.
* **Аналитика и визуализация (EDA):** Скрипт `dashboard.py` выгружает логи из БД через `Pandas`, агрегирует суммарное время в минутах, фильтрует выбросы (группирует редкие программы в категорию "other") и строит дашборд (Bar chart + Pie chart) с помощью `Matplotlib`.
* Desktop GUI (interface.py): Графический интерфейс на CustomTkinter с отображением топ-3 приложений и круговой диаграммы в реальном времени.

## 🛠 Стек технологий

* **Язык:** Python
* **Data Engineering:** `sqlite3`, `json`
* **Data Analysis & Viz:** `pandas`, `matplotlib`
* **System API:** `psutil`, `pywin32` (`win32gui`, `win32process`)
* GUI: customtkinter, Pillow (для работы с иконками и изображениями)
* Dashboard: streamlit (веб-дашборд)

## ⚙️ Установка и запуск

### 1. Установка зависимостей

```bash
pip install pandas matplotlib psutil pywin32 customtkinter pillow streamlit
```

### 2. Сбор данных (обязательно первый запуск)

Запустите трекер активности. Скрипт работает в фоновом режиме и записывает данные при смене активного окна:

```bash
python main.py
```

Для остановки нажмите `Ctrl+C`. Данные сохраняются в `data/screen_time.db`.

### 3. Просмотр статистики (на выбор)

**Desktop GUI (рекомендуется):**
```bash
python interface.py
```

**Веб-дашборд:**
```bash
streamlit run dashboard.py
```

**Быстрая проверка (консоль):**
```bash
python test.py
```

## 📊 Как это работает

1. `main.py` опрашивает активное окно каждую секунду через Windows API
2. При смене окна сохраняется сессия: процесс, заголовок, время начала/конца, длительность
3. Названия процессов нормализуются через `config/process_names.json` (неизвестные добавляются автоматически)
4. `interface.py` / `dashboard.py` читают данные из БД и строят визуализации
